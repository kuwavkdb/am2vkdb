# Amazon Product Evaluator for VKDB

このChrome拡張機能は、[vkdb.jp](https://www.vkdb.jp/) にAmazonの商品情報を素早く掲載するために開発されました。
Amazonの商品一覧ページおよび詳細ページで、商品の評価（○/✗）や著者情報の抽出を行うことができます。

## 主な機能

### 1. 商品情報の抽出とコピー
商品詳細ページ（CD、DVDなど）に、vkdb.jpへの登録に適した形式で商品情報を表示するテキストエリアを追加します。
*   **ワンクリックコピー**: テキストエリアをクリックすると、内容がクリップボードにコピーされます。
*   **カレンダー登録リンク**: 商品の発売日が取得できた場合、vkdb.jpのカレンダー編集ページへのリンク（日付指定済み）を自動生成します。
*   **フォーマット設定**: 出力フォーマットは設定画面で自由にカスタマイズ可能です（例: `{{aitem [[asin]],[[title]],...}}`）。

### 2. 商品・著者の評価管理
Amazon上の商品を効率的に選別するための評価機能を提供します。
*   **商品評価**: 商品画像上の「○」「✗」ボタンで、Good/Badを判定。
    *   **Good (○)**: 商品が緑色の枠で強調表示されます。
    *   **Bad (✗)**: 商品がグレーアウトされ、目立たなくなります。
*   **著者評価**: 商品タイトルの著者名横に評価ボタンを表示。
    *   **Good (○)**: その著者の名前がハイライトされ、著者の商品（未評価のもの）も強調表示されます。
    *   **Bad (✗)**: その著者の名前が取り消し線で表示され、著者の商品（未評価のもの）はグレーアウトされます。商品詳細ページでは、抽出テキストエリアも非表示になります。

### 3. 設定画面
拡張機能のアイコンをクリックして「オプション」を選択すると、以下の設定が可能です。
*   **著者リスト管理**: Good/Badに登録された著者を確認・追加・削除できます。
*   **出力フォーマット設定**: テキストエリアに表示する文字列のテンプレートを設定できます。
    *   使用可能なプレースホルダー: `[[asin]]`, `[[title]]`, `[[author]]`, `[[date]]`, `[[image_url]]`
*   **Date Link設定**: カレンダー登録用リンクのベースURLを変更できます。

## インストール方法
1.  このリポジトリをクローンまたはダウンロードします。
2.  Chromeブラウザで `chrome://extensions/` を開きます。
3.  右上の「デベロッパーモード」をオンにします。
4.  「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダを選択します。

## 開発者向け情報
*   **Manifest V3**: 最新のChrome拡張機能仕様に準拠しています。
*   **権限**: `storage`（設定の保存）, `activeTab`, `https://www.amazon.co.jp/*`（スクリプトの実行と著者情報の取得）
